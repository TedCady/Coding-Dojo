1. Create new project
	1.1. mkdir <projectName>		<-creates project directory
	1.2. cd <projectName> 			<-move terminal location into new directory
	1.3. npm init -y 			<-create package.json for server
	1.4. npm install mongoose express	<-installs dependencies
	1.5. touch server.js			<-create server javascript file

		<CODE>
		const express = require('express');
		const app = express();
		const port = 8000;

		app.listen(port, () => console.log(`Listening on port: ${port}`));
		</CODE>

	1.6. mkdir server			<-create server directory, use mkdir for all children directories as well
		/config
		/controllers
		/models
		/routes
	1.7. npx create-react-app client	<-initiate react project
